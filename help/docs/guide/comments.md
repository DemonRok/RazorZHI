# Comments Overview

You can use `//` or `#` at the beginning of the line to leave comments in your script.  These lines will be ignored during execution.

## // or \# 

!!! example

    === "Using //"

        ```razor
        // Hello world example
        say 'hello'
        //say 'world'
        ```

    === "Using #"

        ```razor
        # Hello world example
        say 'hello'
        # say 'world'
        ```
